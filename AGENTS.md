# Skills

This repo includes Codex-native replacements for the legacy `.claude/commands/*` workflows.

## Available skills

- `comms-inbox-triage`: Daily Gmail triage workflow using Comms MCP tools. (`.codex/skills/comms-inbox-triage/SKILL.md`)
- `comms-weekly-calendar-review`: Start-of-week calendar review + conflict handling. (`.codex/skills/comms-weekly-calendar-review/SKILL.md`)
- `comms-daily-meeting-prep`: Pull meeting context from Calendar, Grain, Gmail, and Slack. (`.codex/skills/comms-daily-meeting-prep/SKILL.md`)
- `comms-meeting-followups`: Post-meeting follow-ups, including interview handling via Ashby. (`.codex/skills/comms-meeting-followups/SKILL.md`)

## Trigger guidance

- Use these skills when the user asks for inbox triage, weekly calendar review, meeting prep, or meeting follow-ups in this repo.
- If the user names one of the skills directly, use that skill.
